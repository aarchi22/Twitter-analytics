# Twitter Analytics Dashboard

## Description
This project is created using Microsoft Power BI.
It analyzes Twitter data to understand user engagement such as likes, retweets, replies, and trends.

## Features
- Top 10 tweet analysis
- Engagement analysis using scatter chart
- Media vs engagement comparison
- Time-based filtering
- Multiple charts like bar, line, column, and dual-axis

## Tools Used
- Microsoft Power BI
